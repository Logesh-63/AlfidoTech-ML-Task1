AlfidoTech Machine Learning Internship – Task 1
Supervised Classification Model

This repository contains the implementation of Task 1 for the Machine Learning Internship at AlfidoTech.

Objective

Build and evaluate a supervised classification model using machine learning algorithms and compare their performance.

Dataset

Breast Cancer Dataset from Scikit-learn

Total Samples: 569

Features: 30

Target Classes:

0 → Malignant

1 → Benign

Algorithms Used

The following machine learning algorithms were implemented:

Logistic Regression

Random Forest Classifier

Steps Performed

Data preprocessing

Train/Test split

Feature scaling using StandardScaler

Model training

Cross-validation

Model evaluation

Evaluation Metrics

The models were evaluated using:

Accuracy

Precision

Recall

F1 Score

ROC-AUC

Tools and Technologies

Python

Google Colab

Pandas

NumPy

Scikit-learn

Matplotlib

Results

Both models performed well on the dataset. However, Random Forest showed slightly better performance due to its ability to capture complex patterns in the data.
